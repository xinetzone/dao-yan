# Feature: 今天你用心了嘛？— 修仙养成世界

## Context
添加一个独立的"修仙养成"游戏功能页面，融合帛书版道德经、佛家直心如如不动、万物理论（Theory of Everything）三大思想体系，设计成每日打卡 + AI 点拨 + 境界养成的沉浸式体验。

---

## 页面入口
- 在 `Index.tsx` 右上角按钮区添加一个带有 `Flame` (lucide) 图标的入口按钮 **"今天你用心了嘛？"**
- 点击跳转至 `/cultivate` 路由（使用 `useNavigate`）

---

## 新增路由
`src/router.tsx` → 添加 `{ path: "/cultivate", element: <CultivationPage /> }`

## 数据层：`src/hooks/useCultivation.ts`

| 境界 | EP 门槛 | 颜色 |
|------|---------|------|
| 凡人 | 0 | 灰 |
| 炼气 | 50 | 绿 |
| 筑基 | 200 | 蓝 |

| 元婴 | 1500 | 橙 |

| 合体 | 12000 | 粉 |
| 大乘 | 40000 | 红 |
| 渡劫 | 120000 | 靛 |
| 真仙 | ∞ | 纯金 |

### 持久化
- 存储于 `localStorage`（`key: "cultivation_state_v1"`），无需登录
- 数据结构：`{ enlightenmentPoints, checkInStreak, totalCheckIns, lastCheckInDate, records[] }`

### 每日打卡积分规则
```
基础: 10
心境加成: 通透+15 / 宁静+10 / 波动+5 / 纷乱+3
无为指数: score × 4 (0-5)
道场感应: +10
感悟字数: >20字 +8 / >5字 +4
连续加成: min(streak × 2, 20)
```

### AI 点拨（复用 `fetchEventSource` 模式）
- 使用与 `useAIChat.ts` 相同的 SSE 调用方式，调用 `ai-chat-167c2bc1450e` edge function
- System prompt 设定：身为通晓帛书版道德经、佛家直心、量子场论的高阶仙师
- 回应结构：① 帛书原文引用 ② 佛家直心点评 ③ 万物理论宇宙视角
- 200字内，古雅诗意

## UI 层：`src/pages/CultivationPage.tsx`

### 整体视觉
- 背景：深空黑蓝 `#060a14`，CSS 动态星点
- 元素：灵气球（按境界颜色发光）+ 进度环 + 描述文字
- 字体：加大字重、增加字间距营造仙气

### 视图状态机：`'home' | 'checkin' | 'result' | 'records'`

#### home 视图（主界面）
- 顶部导航：← 返回 + 标题
- 中央：灵气球（脉冲动画）+ 当前境界名 + 境界描述
- 进度条：当前 EP / 下阶门槛
- 统计行：总打卡数 / 连续天数 / 悟道点
- 操作按钮：今日打卡（已打卡则灰显）/ 修行记录

#### checkin 视图（打卡问答，4步）
1. **心境** — 4 个选项卡（通透/宁静/波动/纷乱），每项有小字描述
2. **无为指数** — 5 星评分点击组件，配经典释义
3. **道场感应** — 开关切换，Yes/No 配哲理提示
4. **心言自述** — 多行文本框（选填）
- 底部：提交按钮 → 调用 AI，显示 loading 动画（"仙师正在感应天机…"）

#### result 视图（结果）
- 大字动画显示「+XX 悟道点」
- 升阶提示（若跨越 EP 门槛）
- AI 仙师点拨全文（渐入动画）
- 按钮：返回主界面

#### records 视图（历史）
- 最近 30 条记录卡片列表
- 每条：日期 / 心境标签 / 积分 / AI 点拨折叠展开
