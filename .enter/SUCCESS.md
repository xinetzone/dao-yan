### 3. 创建 Release 发布 🚀

1. 点击右侧 **"Releases"**
2. 点击 **"Create a new release"**
3. 选择标签：**v1.0.0**
4. Release title：`v1.0.0 - 道研助手首次发布 / Initial Release`
5. 描述（可复制以下内容）：

```markdown
# 🎉 道研助手 v1.0.0 - 首次发布

## ✨ 核心特色

道研助手是一款融合传统哲学智慧与现代 AI 技术的创新应用，旨在帮助用户在信息洪流中找到内心的宁静与智慧。

### 主要功能

1. **🤖 AI 研究助手**
   - 基于 Claude Sonnet 4.5 的智能对话
   - SSE 流式响应，实时展示思考过程
   - 支持思考链可视化

2. **🧘 修炼系统**
   - 10 个修炼境界（凡人 → 真仙）
   - 4 种心境状态（通透、宁静、波动、纷乱）
   - 每日打卡获得悟道点
   - AI 仙师个性化指导
   - 新手教程（5 步互动式引导，50 悟道点奖励）

3. **🌐 联网搜索**
   - DuckDuckGo 免费搜索集成
   - AI 自动判断何时需要搜索
   - 来源引用卡片展示
   - 可选启用/禁用

4. **📚 文档集合管理**
   - 支持 7 种文件格式（TXT, MD, HTML, JSON, CSV, XML, YAML）
   - 网页内容自动抓取
   - 多集合管理
   - 作为 AI 对话上下文

5. **🌍 完整国际化**
   - 中英文双语支持
   - 自动语言检测
   - 实时切换

## 🛠️ 技术栈

**前端**：
- React 19.1 + TypeScript 5.9
- Vite 7 + Tailwind CSS 3.4
- shadcn/ui 组件库
- React Router v7
- i18next

**后端**：
- Supabase（数据库 + Edge Functions）
- PostgreSQL
- Deno（Edge Functions 运行时）

**AI & 搜索**：
- Claude Sonnet 4.5
- DuckDuckGo Web Search

## 🚀 快速开始

### 安装依赖
\`\`\`bash
pnpm install
\`\`\`

### 配置环境
1. 复制 `.env.example` 为 `.env`
2. 配置 Supabase 连接信息
3. 配置 AI API Token

### 启动开发服务器
\`\`\`bash
pnpm dev
\`\`\`

详细说明请查看 [README.md](./README.md)
```
