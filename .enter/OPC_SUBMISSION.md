## 描述

### 我构建了什么

道衍是一款以帛书版《道德经》为知识根基的 AI 研究对话工具。它将两千年前的古老智慧与现代 AI 技术深度融合，提供四大核心能力：

- **原文解读**：精准解析帛书版《道德经》原文，区分帛书版与通行本的关键差异（如"非恒道"vs"非常道"）
- **智能联网搜索**：一键启用 DuckDuckGo 实时搜索，AI 自动提取、综合网络信息辅助研究
- **文档集合系统**：上传 TXT/MD/HTML/JSON 等文档作为对话上下文，支持多集合管理
- **修炼打卡系统**：游戏化的个人成长体系，10 个修炼境界（凡人到真仙）、4 种心境状态、AI 仙师个性化点拨

技术亮点：
- 双主题设计系统：学术暖米色（宣纸底+朱红）+ 修炼深色宇宙（磨砂玻璃+星空）
- 流式响应 + 思考过程可视化（Claude Sonnet 4.5 via SSE）
- Markdown 富文本渲染（marked + highlight.js + DOMPurify）
- 中英双语国际化

### 面向谁

1. **道家文化研究者**：需要精准帛书版原文解读和版本对比的学者、爱好者
2. **哲学探索者**：对东方哲学与现代科学（量子场论）交叉领域感兴趣的思考者
3. **身心修行者**：希望将古老智慧融入日常生活、追踪个人成长的修行人
4. **传统文化教育者**：需要一个互动式教学工具来讲解道家思想的老师

### 如何商业化

1. **知识付费模式**：
   - 免费层：基础对话 + 每日 5 次提问
   - 专业版（29元/月）：无限对话 + 联网搜索 + 文档上传 + 修炼系统
   - 学者版（99元/月）：全部功能 + 优先响应 + 专属古籍数据库

2. **B2B 教育授权**：
   - 向大学国学院、传统文化教培机构提供 API 授权或白标定制
   - 按学生数/座席数收费

3. **内容变现**：
   - 整理 AI 生成的高质量解读为付费电子书/课程
   - 修炼系统数据分析报告（个人成长洞察）

4. **文化 IP 衍生**：
   - "大师兄"角色 IP 授权
   - 道家主题周边（书签、手账本、冥想音频）
