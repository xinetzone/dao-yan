### 3. 创建 Release 发布 🚀

1. 点击右侧 "Releases" → "Create a new release"
2. 选择标签：`v1.0.0`
3. Release title: `v1.0.0 - 道衍首次发布`
4. 描述：

```markdown
# 道衍 v1.0.0 - 首次发布

## ✨ 核心特色

- 🤖 **AI 研究助手** - 基于 Claude Sonnet 4.5 的智能对话
- 🧘 **修炼系统** - 融合道家、佛家智慧的个人成长游戏
- 🌐 **联网搜索** - DuckDuckGo 免费搜索集成
- 📚 **文档上下文** - 本地文件和网页内容支持
- 🌍 **双语支持** - 完整的中英文国际化

## 🛠️ 技术栈

- React 19 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (数据库 + Edge Functions)
- Claude Sonnet 4.5 API

## 📦 主要功能

1. **AI 对话** - SSE 流式响应，思考过程可视化
2. **修炼打卡** - 10 个境界，每日打卡获得悟道点
3. **文档集合** - 上传文件或抓取网页作为 AI 上下文
4. **联网搜索** - 实时搜索网络获取最新信息
5. **新手教程** - 5 步互动式引导，50 悟道点奖励

## 🚀 快速开始

查看 [README.md](./README.md) 获取详细的安装和使用指南。

**愿道衍帮助每一位使用者，在信息洪流中找到属于自己的那份宁静与智慧。** 🌟
```
